# 书籍列表

## 项目说明

可以存放书籍，并添加同时可以选择书籍的阅读的状态

## 字段设计

首先有一个name字段，用来存放书籍的名字，然后又author字段用来存放作者名字，还有status字段用来存放书籍的阅读状态

## 核心功能

- 实现了书籍的添加
- 书籍的删除

## 实现思路

- 每次数据变动时，先用 innerHTML = '' 把旧的列表清空。  
- 使用 forEach 扫描数组，每发现一本书，就动态创建一个 ```<li>``` 标签。  
- 在生成 HTML 时，通过 ${index} 将数组的序号绑定到按钮的 onclick 事件上，确保点击“删除”时程序知道该删哪一行。

## 遇到的问题
添加的时候总是把我最初的那两本数添加上去
解决方案，是每次渲染的时候没有使用 ```innerHTML = ''```来清空文档
